Pyjamas

Theme for VS Code

Pyjamas is a set of dark themes for VS Code. Consisting of three themes - default, blue, and purple - and a wide range of syntax selectors, Pyjamas has something for everyone no matter the codebase.

Use the selector at the top of the page to try out the different themes.

Installation Options

  1. Install through the VS Code Marketplace
  2. Install through the VS Code app
  3. Build from source in GitHub

Blue
